Pulls items from the inventory it is pointed at and places them into the [ME Network](../ME- Network/index.html)'s Networked Storage. You can specify which items it will pull out via the UI, else it tries to pull out any item in the adjacent inventory. The ME Item Import Bus will attempt to import any possible options, even if 1 or more of the configured items cannot be stored. The ME Item Import Bus requires a channel to function.

This is the functional opposite of the ME Item Export Bus.
