Network Tool

Tool that can be used to remove parts from cables ( like any other BC compatible Wrench ), and has a 9 slot inventory that can store AE Upgrade Cards. When in your inventory the inventory of the Network Tool appears in machine GUIs which accept upgrades.
When right clicked on any network component it will give you break down of all the attached parts, and power storage / usage details for the network.
